# Marketing-ab-test-ad-vs-psa
Analisis A/B testing pada dataset marketing_AB untuk membandingkan conversion rate antara grup iklan “ad” dan “psa”. Meliputi perumusan hipotesis, uji signifikansi (two-sided t-test), perhitungan conversion rate, estimasi sample size (power analysis), serta rekomendasi strategi berdasarkan hasil.
